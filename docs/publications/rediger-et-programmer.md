# Rédiger et programmer une publication

1. **Donne un titre** à ta publication pour pouvoir la retrouver plus facilement parmi toutes tes publications

2. **Rédige le contenu** de ton post (tu peux utiliser le formatage proposé : gras, italique, souligné, barré) et prévisualise à gauche comment il sera affiché sur les différents types d'appareils (PC, tablette, mobile)

3. **Ajoute un média** de ton choix : image, document, vidéo

4. **Ajoute des commentaires épinglés** : tu peux ajouter jusqu'à 3 commentaires épinglés à ta publication qui seront publiés instantanément après la publication de ton post

5. **Choisis le statut de ton post** :
   - **Brouillon** : si c'est simplement une idée
   - **Immédiatement** : si tu veux poster tout de suite
   - **Programmer pour plus tard** : pour le programmer à une date précise

6. Une fois terminé, clique sur **Valider**

![Rédiger une publication](/img/publications/rediger-step1.png)

:::info
Pour le moment, il n'est possible que d'ajouter un seul média (peu importe le type).
:::
