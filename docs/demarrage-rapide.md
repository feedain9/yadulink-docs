# Démarrage rapide

Configurez Yadulink en 5 minutes et lancez votre première automatisation.

## Étape 1 : Installer l'extension Chrome

1. Rendez-vous sur le [Chrome Web Store](https://chrome.google.com/webstore)
2. Recherchez "Yadulink"
3. Cliquez sur **Ajouter à Chrome**
4. Confirmez l'installation

:::info Note
L'extension Chrome est nécessaire pour que Yadulink puisse interagir avec LinkedIn de manière sécurisée.
:::

## Étape 2 : Connecter votre compte LinkedIn

1. Connectez-vous à [app.yadulink.com](https://app.yadulink.com)
2. Allez dans **Paramètres** > **Compte LinkedIn**
3. Cliquez sur **Connecter LinkedIn**
4. Autorisez la connexion

## Étape 3 : Créer votre première liste

1. Rendez-vous sur LinkedIn Sales Navigator
2. Effectuez une recherche de prospects
3. Cliquez sur l'icône Yadulink dans votre navigateur
4. Sélectionnez **Ajouter à une liste**

## Étape 4 : Lancer une automatisation

1. Dans Yadulink, allez sur **Automatisations**
2. Cliquez sur **Nouvelle automatisation**
3. Choisissez votre liste de prospects
4. Configurez vos étapes :
   - Visite de profil
   - Demande de connexion
   - Message de suivi
5. Lancez l'automatisation !
