# Comment générer une réponse à un message

Répondez rapidement avec l'aide de l'IA.

## Utilisation

1. Ouvrez une conversation dans l'inbox
2. Cliquez sur l'icône **IA**
3. L'IA génère une suggestion de réponse
4. Modifiez si nécessaire
5. Envoyez

## Personnalisation

L'IA s'adapte à votre ton et style de communication habituel.

## Bonnes pratiques

- Relisez toujours avant d'envoyer
- Personnalisez les détails spécifiques
- Gardez un ton authentique
