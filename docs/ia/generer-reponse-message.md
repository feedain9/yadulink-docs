# Générer une réponse à un message

Les réponses IA se génèrent depuis l'inbox Yadulink. L'ancienne génération dans l'extension n'est plus utilisée.

## Comment générer une réponse

1. Ouvre **Yadulink > Inbox**.
2. Synchronise ta messagerie LinkedIn si Yadulink le demande.
3. Ouvre la conversation de ton choix.
4. Utilise l'assistance IA pour préparer une réponse.
5. Relis, ajuste le message et envoie-le depuis Yadulink.

Si la conversation n'apparaît pas, relance la synchronisation de l'inbox ou reconnecte LinkedIn depuis la bannière affichée.
