# Générer un premier message via le profil

Créez des messages d'approche personnalisés basés sur le profil du prospect.

## Comment ça marche

1. Consultez le profil d'un prospect
2. Cliquez sur **Générer un message**
3. L'IA analyse le profil et génère un message personnalisé
4. Ajustez selon vos besoins

## Éléments analysés

- Poste actuel
- Expérience professionnelle
- Publications récentes
- Points communs avec vous

## Conseils

- Mentionnez un élément spécifique du profil
- Proposez de la valeur dès le premier message
- Restez concis et professionnel
