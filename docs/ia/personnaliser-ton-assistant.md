# Personnaliser ton assistant IA

Configurez l'IA pour qu'elle corresponde à votre style.

## Paramètres de personnalisation

### Ton de communication

- Formel
- Décontracté
- Professionnel amical

### Langue

Choisissez la langue par défaut de vos messages.

### Informations sur vous

Ajoutez des détails sur :
- Votre entreprise
- Vos services
- Votre proposition de valeur

## Instructions personnalisées

Ajoutez des instructions spécifiques pour guider l'IA dans la génération de contenu.
