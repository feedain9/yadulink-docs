# Connecter votre compte LinkedIn

Liez votre compte LinkedIn à Yadulink.

## Prérequis

- Un compte LinkedIn actif
- L'extension Chrome Yadulink installée

## Étapes de connexion

1. Installez l'extension Chrome Yadulink
2. Connectez-vous à LinkedIn dans votre navigateur
3. L'extension détecte automatiquement votre session
4. Vérifiez le statut dans **Paramètres** > **LinkedIn**

## Résolution de problèmes

### Session expirée

Reconnectez-vous à LinkedIn et rafraîchissez Yadulink.

### Extension non détectée

Vérifiez que l'extension est activée dans Chrome.
