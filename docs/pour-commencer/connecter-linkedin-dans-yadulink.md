# Connecter LinkedIn dans Yadulink

L'extension Chrome n'est plus nécessaire. La connexion LinkedIn se fait directement dans l'application Yadulink.

Cette connexion débloque :

- la messagerie LinkedIn et les réponses assistées par IA ;
- les imports de contacts depuis LinkedIn et Sales Navigator ;
- les feeds personnalisés ;
- les automatisations et séquences.

## Depuis l'onboarding

1. Ouvre **Yadulink > Onboarding**.
2. Clique sur **Connecter LinkedIn**.
3. Connecte-toi à LinkedIn dans la page sécurisée affichée par Yadulink.
4. Termine la vérification si LinkedIn demande un code ou une validation.
5. Reviens sur Yadulink : le statut passe en **Synchronisé**.

## Depuis un compte déjà créé

1. Ouvre **Paramètres > Système**.
2. Clique sur **Synchroniser** ou **Reconnecter LinkedIn**.
3. Finalise la connexion dans la page dédiée.

Si une bannière rouge apparaît dans Yadulink, utilise son bouton **Reconnecter LinkedIn** : il te renvoie vers la même page et te ramène ensuite sur l'écran où tu travaillais.

## Bonnes pratiques

- Utilise le compte LinkedIn avec lequel tu veux envoyer les messages et automatisations.
- Garde l'onglet Yadulink ouvert jusqu'à la confirmation.
- Si LinkedIn demande une vérification, valide-la immédiatement depuis la page de connexion Yadulink.
