# L'extension Chrome n'est plus nécessaire

Cette page est conservée pour les anciens liens de documentation.

Yadulink ne fonctionne plus avec l'ancienne extension Chrome : les imports, la messagerie, les feeds et les automatisations ont été déplacés directement dans l'application.

Pour continuer, suis le guide à jour : [Connecter LinkedIn dans Yadulink](./connecter-linkedin-dans-yadulink).

Si tu étais bloqué sur une ancienne étape d'installation, ouvre **Paramètres > Système** dans Yadulink puis clique sur **Synchroniser** ou **Reconnecter LinkedIn**.
