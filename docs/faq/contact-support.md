# Contacter le support

Besoin d'aide ? Notre équipe est là pour vous.

## Chat en direct

Utilisez le chat disponible dans l'application pour une réponse rapide.

## Email

Contactez-nous à **support@yadulink.com**

## Réseaux sociaux

- [LinkedIn](https://www.linkedin.com/company/yadulink)
- [YouTube](https://www.youtube.com/@yadulink)

## Avant de nous contacter

Pour nous aider à résoudre votre problème rapidement :

1. Décrivez précisément le problème
2. Incluez des captures d'écran si possible
3. Mentionnez votre navigateur et version
4. Indiquez les étapes pour reproduire le problème
