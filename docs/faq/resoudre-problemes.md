# Résoudre les problèmes courants

Guide de dépannage pour les problèmes fréquents.

## Problèmes de connexion

### Session LinkedIn expirée

1. Connectez-vous à LinkedIn dans un nouvel onglet
2. Rafraîchissez la page Yadulink
3. Vérifiez le statut de connexion

### Extension non détectée

1. Vérifiez que l'extension est installée
2. Activez-la dans chrome://extensions
3. Redémarrez Chrome

## Problèmes d'automatisation

### Les actions ne s'exécutent pas

- Vérifiez que l'automatisation est active
- Vérifiez les horaires configurés
- Assurez-vous que LinkedIn est ouvert

### Erreurs lors de l'exécution

Consultez les logs de l'automatisation pour identifier l'erreur.
