# Paramètres avancés des automatisations

Optimisez vos séquences avec les options avancées.

## A/B Testing

Testez deux versions de vos messages pour optimiser les performances.

### Configuration

1. Dans une étape de message, activez **A/B Testing**
2. Rédigez la **Version A** et la **Version B**
3. Définissez la répartition (par défaut 50/50)

### Analyse des résultats

Yadulink mesure :
- Taux de réponse par version
- Taux d'acceptation (pour les invitations)

## Filtres conditionnels

Excluez certains prospects de votre automatisation.

### Filtres disponibles

- **Déjà connecté** - Exclure les relations existantes
- **Déjà contacté** - Exclure les prospects déjà approchés
- **Poste** - Filtrer par titre de poste
- **Entreprise** - Filtrer par taille ou secteur
- **Localisation** - Filtrer par région

## Horaires d'exécution

Configurez quand vos actions s'exécutent.

### Plages horaires

- Définissez les heures de début et fin
- Choisissez les jours de la semaine
- Le fuseau horaire est automatiquement détecté

### Bonnes pratiques

- Privilégiez les heures de bureau (9h-18h)
- Évitez les weekends pour la prospection B2B
- Espacez vos actions sur plusieurs heures
