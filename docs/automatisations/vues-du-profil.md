# Vues du profil

Lorsque quelqu'un visite ton profil, c'est un signe d'intérêt implicite. Automatiser l'ajout et l'engagement de ces visiteurs transforme une simple curiosité en opportunité concrète : tu disposes déjà d'un point d'accroche pour personnaliser ton message et démarrer la conversation de manière naturelle.

## Pour configurer cette automatisation

1. Rends-toi dans **Automatisations > Vues du profil**

2. Configure ton message en utilisant la variable `{{first_name}}` pour personnaliser le message au nom du contact

3. Active l'automatisation

Tu peux suivre les profils qui ont été ajoutés et engagés.
