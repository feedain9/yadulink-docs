# Templates de réponse

Créez des réponses types pour gagner du temps.

## Créer un template

1. Accédez à **Messages** > **Templates**
2. Cliquez sur **Nouveau template**
3. Rédigez votre message type
4. Utilisez des variables pour personnaliser

## Variables disponibles

- `{{first_name}}` - Prénom du destinataire
- `{{last_name}}` - Nom du destinataire
- `{{company}}` - Entreprise
- `{{position}}` - Poste

## Utiliser un template

1. Ouvrez une conversation
2. Cliquez sur l'icône templates
3. Sélectionnez votre template
4. Personnalisez si nécessaire
5. Envoyez
