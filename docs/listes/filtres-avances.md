# Filtres avancés

Segmentez vos leads avec précision.

## Filtres disponibles

### Par profil

- Poste / Titre
- Entreprise
- Secteur d'activité
- Localisation
- Taille d'entreprise

### Par interaction

- Connecté / Non connecté
- Message envoyé / Non envoyé
- Réponse reçue / En attente

### Par date

- Date d'import
- Dernière activité
- Date de connexion

## Combiner les filtres

Utilisez les opérateurs ET / OU pour créer des segments précis.

## Sauvegarder vos filtres

Enregistrez vos combinaisons de filtres pour les réutiliser rapidement.
