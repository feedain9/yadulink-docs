# Importer des leads

Importez vos prospects depuis différentes sources.

## Sources d'import

### Recherche LinkedIn

Importez directement depuis une recherche LinkedIn ou Sales Navigator.

### Fichier CSV

Importez depuis un fichier CSV avec les colonnes :
- Prénom
- Nom
- URL LinkedIn
- Entreprise (optionnel)
- Poste (optionnel)

### Autres sources

- Post LinkedIn (commentaires, likes)
- Événement LinkedIn
- Groupe LinkedIn

## Limites d'import

Respectez les limites quotidiennes pour protéger votre compte LinkedIn.
