# Gérer vos leads

Suivez et organisez vos prospects efficacement.

## Vue d'ensemble

Visualisez tous vos leads avec leurs informations clés :
- Nom et poste
- Entreprise
- Statut de connexion
- Dernière interaction

## Statuts personnalisés

Créez des statuts pour suivre la progression :
- À contacter
- En attente
- Intéressé
- Client
- Non intéressé

## Actions en masse

Sélectionnez plusieurs leads pour :
- Les ajouter à une automatisation
- Changer leur statut
- Les déplacer vers une autre liste
- Les exporter
