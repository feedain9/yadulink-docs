# Créer une liste de leads

Organisez vos prospects dans des listes segmentées.

## Créer une nouvelle liste

1. Accédez à **Listes**
2. Cliquez sur **Nouvelle liste**
3. Donnez un nom à votre liste
4. Ajoutez une description (optionnel)

## Types de listes

### Listes statiques

Ajoutez manuellement des leads à votre liste.

### Listes dynamiques

Les leads sont ajoutés automatiquement selon des critères définis.

## Organisation

Utilisez des statuts et tags pour organiser vos leads au sein d'une liste.
