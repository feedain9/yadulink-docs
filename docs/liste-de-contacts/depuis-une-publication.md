# Depuis une publication

1. Depuis ton fil d'actualité (ou depuis un profil), repère une publication dont tu voudrais importer les personnes ayant liké ou commenté la publication. Une fois repéré, clique sur **Importer les profils ayant réagis**.

![Importer les profils ayant réagis](/img/liste-de-contacts/publication-step1.png)

2. La publication vient de s'ouvrir, ouvre maintenant l'extension Yadulink, crée ou choisis une liste de contacts déjà existante, définis le type de contact à importer (1er degré, 2ème degré ou 3ème degré) et clique sur **importer**.

![Importer depuis l'extension](/img/liste-de-contacts/publication-step2.png)

:::info Rappel des différents types de relations sur LinkedIn
- **1er degré** : tes connexions directes, avec qui tu peux échanger librement.
- **2ème degré** : les connexions de tes connexions ; tu peux leur envoyer une invitation ou un message InMail.
- **3ème degré** : les profils hors de ton réseau étendu ; tu peux les trouver via recherche, mais tu ne peux pas les contacter directement sans InMail.
:::
