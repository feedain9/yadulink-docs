# Depuis une publication

Cet import récupère les profils qui ont interagi avec une publication LinkedIn.

1. Ouvre la publication LinkedIn à importer.
2. Copie son URL.
3. Dans Yadulink, va dans **Listes** puis ouvre l'importeur.
4. Colle l'URL de la publication.
5. Choisis la liste de destination.
6. Sélectionne les degrés de relation à inclure.
7. Clique sur **Lancer l'import**.

:::info Rappel des différents types de relations sur LinkedIn
- **1er degré** : tes connexions directes.
- **2e degré** : les connexions de tes connexions.
- **3e degré** : les profils hors de ton réseau étendu.
:::

L'import peut prendre plusieurs minutes selon le volume de réactions.
