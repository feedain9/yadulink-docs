# Depuis un groupe

:::info
Cet import nécessite que tu sois membre du groupe.
:::

1. Ouvre le groupe LinkedIn de ton choix duquel tu veux importer des membres depuis la [page des groupes](https://www.linkedin.com/groups/)

2. Une fois tu as ouvert le groupe, clique sur **Tout afficher** pour afficher tous les membres du groupe

![Afficher les membres du groupe](/img/liste-de-contacts/groupe-step1.png)

3. Une fois cette liste ouverte, ouvre l'extension, crée ou choisis une liste de contacts existante, définis le type de contact à importer (1er degré, 2ème degré ou 3ème degré), puis donne le nombre de contacts que tu souhaites importer (avec une limite de 1000 contacts au maximum) et clique sur **Importer**.

![Importer les membres](/img/liste-de-contacts/groupe-step2.png)

:::info
Cette limite de 1000 contacts est fixée par Yadulink pour éviter le scraping massif de contacts.
:::
