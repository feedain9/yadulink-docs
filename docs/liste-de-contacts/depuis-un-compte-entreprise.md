# Depuis un compte entreprise

Cet import récupère les followers d'une page entreprise LinkedIn.

:::info
Tu dois être administrateur de la page entreprise. Le nouvel importeur attend le lien du tableau de bord admin, pas le lien public de la page.
:::

1. Ouvre la page entreprise depuis LinkedIn.
2. Va dans son tableau de bord admin.
3. Copie l'URL contenant `/company/<id>/admin/dashboard/`.
4. Dans Yadulink, va dans **Listes** puis ouvre l'importeur.
5. Colle l'URL du tableau de bord admin.
6. Choisis la liste de destination.
7. Sélectionne les degrés de relation à inclure.
8. Clique sur **Lancer l'import**.

Si tu colles le lien public d'une page entreprise, Yadulink te demandera de coller le lien du tableau de bord admin.
