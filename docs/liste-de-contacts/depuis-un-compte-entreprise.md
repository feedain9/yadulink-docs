# Depuis un compte entreprise

:::info
Cet import concerne les abonnés d'une page entreprise, mais elle nécessite que tu sois administrateur du compte entreprise.
:::

1. Ouvre ta page entreprise via l'espace à gauche sur LinkedIn

![Accéder à la page entreprise](/img/liste-de-contacts/entreprise-step1.png)

2. Une fois ouvert, ouvre l'extension, crée une liste de contacts ou choisis en une déjà existante, sélectionne le type de contacts que tu veux importer (1er, 2ème ou 3ème degré) et clique sur **Importer**

![Importer les abonnés](/img/liste-de-contacts/entreprise-step2.png)

:::info Rappel des différents types de relations sur LinkedIn
- **1er degré** : tes connexions directes, avec qui tu peux échanger librement.
- **2ème degré** : les connexions de tes connexions ; tu peux leur envoyer une invitation ou un message InMail.
- **3ème degré** : les profils hors de ton réseau étendu ; tu peux les trouver via recherche, mais tu ne peux pas les contacter directement sans InMail.
:::
