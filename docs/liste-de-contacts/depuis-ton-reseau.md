# Depuis ton réseau

Pour importer ton réseau dans une liste de contacts :

1. Ouvre les paramètres des listes de contacts

2. Clique sur **Importer**

![Importer ton réseau](/img/liste-de-contacts/reseau-step1.png)

3. Une fenêtre de confirmation apparaît pour lancer l'import, clique sur **Confirmer**

![Confirmer l'import](/img/liste-de-contacts/reseau-step2.png)

4. L'import est lancé, selon la taille de ton réseau, l'import peut durer de quelques minutes à quelques heures.
