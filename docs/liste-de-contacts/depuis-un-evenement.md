# Depuis un événement

L'import depuis les participants d'un événement LinkedIn n'est pas disponible dans le nouvel importeur Yadulink.

L'ancien parcours passait par l'extension Chrome. Depuis la migration dans l'application, l'importeur web prend en charge :

- les profils LinkedIn ;
- les réactions d'une publication ;
- les recherches LinkedIn ;
- les groupes LinkedIn ;
- les pages entreprise dont tu es administrateur ;
- Sales Navigator.

## Alternatives recommandées

- Utilise une recherche LinkedIn avec les mots-clés de l'événement.
- Importe les personnes ayant réagi à une publication liée à l'événement.
- Si l'événement est associé à une communauté, importe les membres du groupe LinkedIn correspondant.

Cette page est conservée pour clarifier la migration des anciens liens.
