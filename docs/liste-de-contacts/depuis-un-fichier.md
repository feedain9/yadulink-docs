# Depuis un fichier

1. Dans Yadulink, rends toi sur **Listes de contacts** et clique sur **Créer une liste**

![Créer une liste de contacts](/img/liste-de-contacts/fichier-step1.png)

2. Une fenêtre de dialogue s'ouvre pour te demander de remplir le **Nom de la liste de contacts** (tu peux mettre ce que tu veux)

![Nom de la liste](/img/liste-de-contacts/fichier-step2.png)

3. Ensuite, coche **Oui** pour **Import des contacts depuis un fichier .csv** et importe ton fichier en cliquant sur **Choisir un fichier**

![Import depuis un fichier CSV](/img/liste-de-contacts/fichier-step3.png)

4. Une fois le fichier importé, Yadulink pré-remplit les champs qu'il a détecté, assure toi qu'il corresponde bien aux noms de colonnes respectifs de ton fichier et clique sur **Enregistrer**

![Mapping des colonnes](/img/liste-de-contacts/fichier-step4.png)

:::info
Les colonnes dans un fichier .csv se trouvent à la première ligne
:::

### Colonnes disponibles

- **Colonne contenant les noms** : lastName
- **Colonne contenant les prénoms** : firstName
- **Colonne contenant les postes** : occupation
- **Colonne contenant les liens LinkedIn** : inviteeProfileUrl

:::note
Selon le nombre de lignes dans ton fichier, l'import peut prendre jusqu'à quelques minutes.
:::
