# Exporter une liste de contacts

:::info
L'export est uniquement possible au format .csv
:::

1. Dans Yadulink, rends toi sur **Listes de contacts > Paramètres**

2. Une fois dans tes paramètres de listes de contacts, sélectionne la liste que tu veux exporter, clique sur le menu puis sur **Exporter cette liste**

3. Au clic sur **Exporter cette liste**, une fenêtre de téléchargement s'ouvre pour enregistrer le fichier sur votre appareil
